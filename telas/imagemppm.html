Imagem PPM

A imagem ppm é basicamente uma imagem escrita em texto puro, sem formatação, sem JPG, sem PNG, sem nada. Esse formato PPM (Portable Pixmap) é um dos formatos mais simples do mundo. A estrutura segue sendo:

  • Linha 1: significa que os pixels são armazenados como números em texto.
  • Linha 2: largura 3 pixels, altura 3 pixels → ou seja, uma imagem 3x3.
  • Linha 3: igual no RGB clássico.
  • Linha 4: Sempre na ordem R G B R G B R G B… Cada 3 números representa 1 pixel.

O código vai abrir o arquivo e ler seu conteúdo, sendo linha a linha. Então ele vai ter o tipo dele, as dimensões e valores RGB da imagem, então após ir lendo e lendo ele monta no terminal com pixels legíveis o que a imagem ppm significa.

Leitura de arquivo PPM

package main

	import (
		"bufio"
		"fmt"
		"os"
	)

	func main() {
		img, _ := os.Open("image.ppm")

		reader := bufio.NewReader(img)

		var formato string
		var linhas, colunas, maxVal int

		fmt.Fscan(reader, &formato)
		fmt.Fscan(reader, &linhas, &colunas)
		fmt.Fscan(reader, &maxVal)

		fmt.Printf("Formato: %s\n", formato)
		fmt.Printf("Dimensões: %dx%d\n", linhas, colunas)
		fmt.Printf("Valor máximo: %d\n", maxVal)
		fmt.Printf("Pixels:\n")

		var r, g, b int

		for i := 0; i < linhas; i++ {
			for j := 0; j < colunas; j++ {
				fmt.Fscan(reader, &r, &g, &b)
				_ = r
				_ = g
				_ = b
				fmt.Printf("%d %d %d\n", r, g, b)
			}
		}
	}

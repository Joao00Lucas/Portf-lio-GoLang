Características da Linguagem

Principais características da linguagem

  • Característica 1: Compilada - foco no desempenho
  • Característica 2: Segurança - compilando ela antes, evita falhas críticas.
  • Característica 3: Concorrência - suportando nativamente a concorrência entre os processos.
  • Característica 4: Microsserviços - permite ser usada em sistemas distribuídos e em cloud.
  • Característica 5: Backend - foco total nessa modalidade de desenvolvimento.

Documentação e recursos adicionais podem ser encontrados na página oficial da linguagem.

Durante a aula o grupo feito por mim, Maysson e Victor Azam apresentou as questões de simplicidade e desempenho da linguagem Go para a turma, no que cada uma é melhor, colocando ela contra o Python.

Argumentos Usados

  • Go é complicada para linguagem de máquina diretamente, quando o python precisa de interpretador, sendo Go mais rápida por conta disso, simplificando deploy, minimizando erros e dependência;
  • Go usa múltiplos núcleos no processamento, o que ajuda na execução;
  • Go organiza os pacotes de forma importada, o que deixa o código limpo e gerenciável;
  • Python permite uma estrutura mais simples e fácil que Go, usando menos código e sendo mais livre;
  • Python tem aplicação em diversas área como vantagem, por ser mais "maleável".
